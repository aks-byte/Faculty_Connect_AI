# FacultyConnect AI 🎓
AI-powered college portal built at a hackathon.

## Features
- 📅 Live faculty availability + slot booking
- 💬 AI chat with each professor (unique personality per faculty)
- ✉️ AI email generator with tone analyzer
- 🚩 Anonymous grievance tracker
- 📊 Analytics dashboard
- 🎉 Confetti + WhatsApp booking confirmation
- 🌗 Dark/light mode

## Live Demo
👉 [facultyconnect-ai.netlify.app](facultyconnectai.netlify.app)
## Setup
1. Open `index.html` in browser
2. Click 🔑 Add Gemini Key
3. Get free key at [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
4. Done — no install, no backend needed
